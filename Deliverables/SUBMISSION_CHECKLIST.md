# Submission Checklist

- [x] **Seamless Transfer**: Agent transfer works (Bob -> Alice -> Bob) without restarting the session.
- [x] **Context Continuity**: New agent receives prior conversation context and summary.
- [x] **Active Agent Label**: Logs and UI display the currently active agent.
- [x] **Transcript Logging**: Full conversation history is maintained in memory.
- [x] **Voice + Text Fallback**: Both modes work (`--text` flag).
- [x] **Clean Machine Setup**: `requirements.txt` is minimal and verified.
- [x] **Environment Variables**: `.env.example` includes all necessary API keys.
- [x] **Documentation**: `README.md` and `DESIGN_NOTE.md` are complete.
